Components

  • HDFS
  • MapReduce

Hadoop Architecture

NameNode

Stores metadata and manages HDFS.

DataNode

Stores actual data blocks.

JobTracker

Schedules MapReduce jobs.

TaskTracker

Executes tasks.

HDFS

HDFS is Hadoop Distributed File System used for fault-tolerant storage. 

hdfs dfs -mkdir /data
hdfs dfs -put file.txt /data
hdfs dfs -ls /

MapReduce

MapReduce consists of Map, Shuffle and Reduce phases.

Input -> Mapper -> Shuffle -> Reducer -> Output

Apache HBase

Distributed NoSQL database built on HDFS.

create 'student','info'
put 'student','1','info:name','Ajay'
scan 'student'

Topics: Architecture, Filters, Prefix Filter, Single Value Column Filter, TTL.

Apache Cassandra

Highly scalable distributed NoSQL database with peer-to-peer architecture. 

CREATE KEYSPACE college
WITH replication =
{'class':'SimpleStrategy',
'replication_factor':3};

CREATE TABLE student(
id int PRIMARY KEY,
name text
);

Topics: Gossip Protocol, Replication, Consistency Levels, Read/Write Paths, Compaction.
